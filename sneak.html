About ChronoSneak

ChronoSneak is a simple web prototype of a turn based stealth puzzle game which features time travel mechanics.

This simplistic version is for testing out theories/ideas about how the game could work. The experience is somewhat lacking.

ChronoSneak Controls

WASD/Arrow Keys
Move the player. Trying to move in a direction you are not facing causes the player to change their facing
Space Bar/Q
Interact with entities. Entities on the same tile as the player will be interacted with; if none, entities on adjacent tiles will be selected instead, based on their facing
Enter/E
Wait; All entities get a turn while the player remains in the same location
F1
Dump entity information for the entity under the cursor to the console. If the entity is a guard, its patrol is outlined. If the entity has triggers, arrows show what entities it will trigger.
F5
Take a screen shot (opens in a new tab/window)
